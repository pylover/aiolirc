Installation¶

Prerequisites¶

Debian¶

$ apt-get install build-essential python3.5-dev

$ pip install cython

From PyPI¶

pip:

$ pip install aiolirc

From Source¶

$ cd path/to/source/directory
$ pip install .

or:

$ python setup.py install

You may use the source folder in-place as a python library in sys.path, so you have to build the C extensions before using the library:

$ cd path/to/source/directory
$ python setup.py build_ext --inplace

Development, editable:

$ cd path/to/source/directory
$ python setup.py build_ext --inplace
$ nosetests
$ pip install -e .

From Github¶

Latest development code:

$ pip install git+https://github.com/pylover/aiolirc.git
